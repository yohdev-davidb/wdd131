DnD Character Forge

Site Name

The site name is DnD Character Forge. This name was selected to represent a platform where players can craft and customize their own Dungeons and Dragons characters, akin to forging them in a smithy.

Site Purpose

The purpose of this site is to provide an interactive character creation tool for Dungeons and Dragons players. Users can select races, classes, abilities, and equipment to build custom characters, and then download or print their character sheets.

Scenarios

  • How can I create a new character for my upcoming DnD campaign?
  • Can I save and export my character sheet in PDF format?

Color Scheme

The color scheme for the site includes:

  • Primary Color: Indigo (#4B0082) - used for headings and navigation bar.
  • Secondary Color: Goldenrod (#DAA520) - used for buttons and accent elements.

Typography

The typography selections are:

  • Heading Font: Crimson Pro - used for all headings to give a medieval feel.
  • Body Font: Roboto - used for body text for readability.
